mac 显示隐藏文件

输入以下命令后，重启finder后可显示隐藏文件：


defaults write com.apple.finder AppleShowAllFiles -bool true

将文件重新隐藏则将后面参数改为false即可。

ffmpeg 添加webm支持（添加libvpx库）

记录一下：
因为要转换成webm格式，所以添加了libvpx库
ffmpeg每次添加库后都要重新编译。
首先下载libvpx
刚开始用的yum


yum install libvpx

后来编译ffmpeg时提示版本太低：

must be >=0.9.1

于是去下载源码编译：链接


./configure
make &&make install

之后，
进入ffmpeg目录


.configure/ —enable-gpl —enable-version3 —enable-nonfree —enable-libfaac —enable-libmp3lame —enable-libx264 —enable-libxvid —enable-shared —enable-libopencore-amrnb —enable-libopencore-amrwb —enable-libvorbis —enable-pthreads —disable-ffserver —disable-ffplay —enable-libvpx
make &&make install

出现如下错误：


/usr/bin/ld: /usr/local/lib/libvpx.a(vpx_codec.c.o): relocation R_X86_64_32 against .rodata.str1.1’ can not be used when making a shared object; recompile with -fPIC
/usr/local/lib/libvpx.a: could not read symbols: Bad value
collect2: ld returned 1 exit status
make: * [libavcodec/libavcodec.so.54] Error 1

进入libvpx目录，重新编译，
configure添加 —enable-shared


./configure  —enable-shared
make &&make install

接着进入ffmpeg目录重新编译安装
ok！

    }

Flow player at IE9 & IE10

IE9 ＆ IE10 在使用flowplayer（版本6）时 出现“Video file not found»”的错误（html5方式）。
将source type改成video/flash代替video/mp4，便可通过了。
话说这种方法可真够坑人的。

<source type="video/flash" src="..."/>

备注：之前没留意，这种方法最终播放的时候ie9&10也变成了flash方式播放了。还不是不可取。

disqus配置多语言界面

disqus是国外有名的社会化评论系统。安装它的wordpress插件后发现多语言时的界面问题。比如说我目前网站是中日文多语言站点，我希望用户选择日文博客的时候，disqus显示日文，用户选择中文中文的时候显示中文的disqus界面。

最终解决方案：
更改插件文件disqus-comment-system/comments.php
在尾部javascript中进行如下更改：
去掉：

config.language = '<?php echo esc_js(apply_filters('disqus_language_filter', '')) ?>
添加： 
<?php $wplang = substr(get_locale(),0,2); ?>
config.language = '<?php echo (!empty($wplang)) ? $wplang :  esc_js(apply_filters('disqus_language_filter', '')) ?>'

先获取wordpress语言编码，因为中文编码是zh_CN,而disqus中文编码是zh，因此简单起见我取了前两个字符串来解决。然后将编码交给disqus处理。

这个问题是解决了，不过还有一个问题。
评论数量链接显示成“ 0 Comments と 0 Reactions”的模样。懒得改了先就这么放着吧。
